# Basic Campaigns

### Overview

In the Campaigns module, select the **Campaigns** option to access the Basic Campaigns dashboard.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FleOYA4vuQK9ThhANTOFU%2FAccessing%20Basic%20Campaigns%20Dashboard.png?alt=media&#x26;token=61b8dd77-f952-436d-94d2-06f3fe137d84" alt=""><figcaption></figcaption></figure>

Using Basic Campaigns, you can send a one-time message to single or multiple contacts. Also, the message type could be either a simple message or a [pre-approved message template](https://github.com/rampwin/rampwin-gitbook-docs/blob/main/broken-reference/README.md) based on the selected channels.

{% hint style="warning" %}
WhatsApp Business Platform has a Customer Service Window (24-hour) in which businesses can send both simple messages and [pre-approved templates](https://github.com/rampwin/rampwin-gitbook-docs/blob/main/broken-reference/README.md). Outside the customer service window, the business would require a message template.
{% endhint %}

### Create a Basic Campaign

To create a basic campaign, click the **Create Campaign** button in the campaigns window.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FeL9As9UZIAP73FKIB4C7%2FCreate%20a%20Campaign.png?alt=media&#x26;token=fcfc3c42-8d68-45ed-8578-78f8f5f4b140" alt=""><figcaption></figcaption></figure>

This will open a window where you'll find a timeline to create a campaign. The steps of the timeline are,

* Select Sender
* Select Contacts
* Select Template
* And Set Schedule

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FqOExTgkBCmch2xl35nqZ%2FSelect%20Sender%20of%20the%20Basic%20Campaign%20Timeline.png?alt=media&#x26;token=d880d98a-7259-4e5a-8e62-e400e100f4e4" alt=""><figcaption><p>Create Campaign Window</p></figcaption></figure>

The first step allows you to set the Campaign Name and select the Channel through which you want to send the campaign message. Simply enter the name, then select the channel, and after that, click **Next**.

### Select Contacts

Upon clicking next, you'll arrive at the second step of creating a campaign which is **Select Contacts**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2Fng8SV4u0q8KfRnoM7Yea%2FCreate%20Campaign%20Select%20Contacts.png?alt=media&#x26;token=5fadd33f-8a67-46e9-b13f-43975eebbd76" alt=""><figcaption><p>Select Contacts Step</p></figcaption></figure>

Now to select the contacts, there are 2 ways.

1. By filtering the existing contacts.
2. By uploading the contacts.

#### Filtering Existing Contacts for the Campaign

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FAwstFd6acjxb7XC3HFq8%2FAccessing%20Filter%20options%20for%20the%20basic%20campaign.png?alt=media&#x26;token=6e628d22-56be-46cc-af14-195aba8768cd" alt=""><figcaption><p>Accessing Filter Options</p></figcaption></figure>

Click the **Filter** button to access the filter options.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FSRMDWPBWkxOavHGxwx3C%2FFilter%20Options.png?alt=media&#x26;token=6a00c201-91fc-43be-b87e-45fa630a1512" alt=""><figcaption><p>Filter Options</p></figcaption></figure>

The filter options available are,

* **Columns**
* **Operator**
* **Value**

First, you need to select the column on the basis you want to filter the contacts. There are 10 options from which you can select the column.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2F2OFFl8A9JvpVmb9wMmKM%2FFilter%20Column%20Options.png?alt=media&#x26;token=53eaabb1-cd81-4fe2-bc37-dd6e5bff9079" alt=""><figcaption></figcaption></figure>

After selecting the column, select the Operator that will verify the Value in the column. There are 9 operator types from which you can select one.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FaqWPJQShFsOaqUU4Yaoo%2FFitler%20Operator%20Types.png?alt=media&#x26;token=6d9bb59d-c2ab-47aa-a2b8-75ea5f672226" alt=""><figcaption></figcaption></figure>

* IS - The Column and Value should exactly match to apply the filter.
* STARTSWITH - The value in the Column should start with the value in the Value field to run the filter.
* CONTAINS - If the value in the Column includes the value in the Value field, the filter will run.
* IS NOT - To apply the filter, the Column field should not match with the Value field.
* DOES NOT CONTAIN - To apply the filter, the Column field doesn't include the instances of the Value field.
* IN - The Tags be IN the Value field
* NOT IN - The Tags be NOT IN the Value field.
* MATCHES - The Channel Name should MATCH the Value.
* BETWEEN - The message should be Created or Updated BETWEEN the Date Range set in the Value.

Once selected the operator, enter the value or select the option available in the Value field. The value field based on Column provides you with different options that are,

* Tag list for the Tag column.
* User / Group for the Contact Type column.
* Channel names for the Channel Name column.
* Date Range (Today, Yesterday, This Week, This Month and Custom) for Created At and Updated At column.

After adding the values in the fields, click **Submit**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FQcZjzRAF4y21Uvd1Sg3s%2FFiltering%20the%20Contacts.png?alt=media&#x26;token=5bc3ed52-7837-4049-91f1-3ef5890dc9d0" alt=""><figcaption><p>Filtering the Contacts</p></figcaption></figure>

As soon as you click the submit button, you'll find the contacts as the following,

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FT2xTlwnTHgertcT766tH%2FFiltered%20Contacts.png?alt=media&#x26;token=e209ff58-b7ca-40a5-9baa-0ce1387783cc" alt=""><figcaption></figcaption></figure>

If you want to add more filter groups, you can click the **Add Filter Group.**

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FF5opqsvzz76Y0WYIVFFL%2FNew%20Filter%20Group.png?alt=media&#x26;token=e00253aa-82a2-44bc-9315-5ba29176fafa" alt=""><figcaption></figcaption></figure>

Or, if you want to add another filter in the same group, click the Plus button ![](https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2F1WNM2opBmG6WLkD1GbcH%2FPlus%20button%20for%20Filters.png?alt=media\&token=fc82bbfa-b811-4196-941e-a41568050bee)

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2Fb5xgiPFKf9qiQ3fED8z9%2FNew%20Filter%20Rule.png?alt=media&#x26;token=98553685-26f7-469e-9733-4438238eee83" alt=""><figcaption></figcaption></figure>

To delete a Filter Group, you can click the **Delete Filter Group** button associated with that group.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FbVztub99DsFutJLie4Ki%2FDelete%20a%20Filter%20Group.png?alt=media&#x26;token=b3e804cd-289a-4d6f-b95c-cbc01c323e0e" alt=""><figcaption></figcaption></figure>

And to delete a filter, you can simply click the Trash button<img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FnOKuiGH16Qhb31eqieuH%2FDelete%20Button%20for%20Filter.png?alt=media&#x26;token=8155fb9e-8453-4566-9a95-842e384b0341" alt="" data-size="original">associated with the particular filter rule. Once you're satisfied with your filtered contacts, you can click the **Next** button.

#### Uploading Contacts for the Campaign

If there are no contacts, or you want to use different contacts other than the existing ones, you can upload contacts for the campaign. And to do that, click **Upload Contacts**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FTz02FuGfQvuK9FmUadMg%2FUploading%20Contacts%20for%20the%20Campaign.png?alt=media&#x26;token=a0454fa7-3631-46b2-bc73-7993a2ca62c1" alt=""><figcaption></figcaption></figure>

A modal will appear with the options to choose the Country Code, Add Tags, and Select the file. If the file already has a country code for the contacts, you can tick the **Phone Number have country code** checkbox.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FmpuuCPfCi0LNGMA1hqlk%2FPhone%20Number%20have%20country%20code.png?alt=media&#x26;token=b5cc00ee-2410-4e46-9267-3c3572f88bf4" alt=""><figcaption></figcaption></figure>

Otherwise, first, select the country code, then add tags (if required), and add the contacts' file from your local system.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2Fej2Elvjm6nmvtXiYu3dp%2FUploading%20the%20file.png?alt=media&#x26;token=1eed9a25-74ee-441d-b015-acb123c2d0da" alt=""><figcaption></figcaption></figure>

As you do that, you'll see your contact numbers and other details in the modal. Once you're satisfied, click **Next**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2Fg77s5i24UwnaKoSjzsFU%2FUploading%20Contacts%20check.png?alt=media&#x26;token=b403bba6-29d7-4735-96c1-e33e72683a25" alt=""><figcaption></figcaption></figure>

On the next screen, ensure that all the information is assigned to the right column. If everything is correct, click **Next**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FVrVfgdfMOa2dVn5ubx99%2FFinishing%20Contacts%20Upload.png?alt=media&#x26;token=f96dd74e-35b3-4fde-ba4e-f29fa13f4683" alt=""><figcaption></figcaption></figure>

It'll take a few moments to upload the contacts, and once the process is completed, you can click **Finish**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2Fv6rguBmGLEbLrAt7yWd6%2FUploaded%20Contacts.png?alt=media&#x26;token=4a1cf0db-c165-4d69-903c-f70300f14edd" alt=""><figcaption></figcaption></figure>

As you finish, you'll find your contacts. Now, you can move to the 3rd step to create a campaign: **Select Template**.

### Select Template

Now, based on your selected Channel, you either get the option to send a template message. Or create a new one.

For example, if you select WhatsApp Channel as 360dialog, Emma, or Cloud API, you can only send a template.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FqcCox9YIp4eZZoNm7rZ7%2FSelect%20a%20Template.png?alt=media&#x26;token=78a065fd-8190-42ed-bfd3-597cdd26dc43" alt=""><figcaption></figcaption></figure>

And if you've selected the Channel as Maytapi, Facebook, or Instagram, you can send a message template or create a new one.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FAZQ10MpdhwadngDRl6q2%2FSelect%20a%20Message%20or%20Create%20a%20New.png?alt=media&#x26;token=106368bc-10f1-48ea-8219-a1a6b2926288" alt=""><figcaption></figcaption></figure>

As per requirement, select the template or an existing message or create a new one. Once you're done, click **Next**.

### Set Schedule

On the **Set Schedule** window, you get to see all the information about your campaign. Plus, the option to send it right away or based on a schedule.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2F5QTYSyivcRcqn5Fmpfiv%2FCampaign%20Preview.png?alt=media&#x26;token=5b681235-04c5-4303-b85f-4525da78bae1" alt=""><figcaption></figcaption></figure>

You can review the Campaign Name, Contacts, and Template as well as preview the content in the template. To send the Campaign the moment you've created it, you can click the **Send Now** button.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FybdMNyTtmo9tgvvAlZwy%2FSend%20Campaign%20Now.png?alt=media&#x26;token=25e2d93f-fd0c-4443-b94b-b2f7a7d00712" alt=""><figcaption></figcaption></figure>

This will run your campaign immediately. Or, if you want to schedule it, you can enter the date and time in the _**Set Schedule**_ field.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2F4UpfnYrgLy8LdJRoPZCA%2FScheduling%20a%20Campaign.png?alt=media&#x26;token=1c8a113c-945d-4840-8b80-bb546f395161" alt=""><figcaption></figcaption></figure>

First, select the date, then define the time and once both are defined, click **Ok**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FGA0onMaYA88Vs3TCzI5v%2FBroadcast%20Campaign.png?alt=media&#x26;token=a0c17724-091b-47cc-90ff-5a8e9faefa58" alt=""><figcaption></figcaption></figure>

To run the campaign at different hours of the day, you can click the checkbox _Do you want to run this campaign in specific duration of day?_

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2F8H5FkUaOHjrbZOEKGMw0%2FSet%20Broadcast%20Time%20and%20click%20Finish.png?alt=media&#x26;token=d70751de-d3e2-425b-a73f-0fa7da1c9f0c" alt=""><figcaption></figcaption></figure>

Then input your Start and End Times for the broadcast. Once done will all the options of the schedule, whether sending it now, scheduling or broadcasting at specific hours, click **Finish**.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FIQH9OQ0tEgmfzinLMamj%2FCampaign%20List.png?alt=media&#x26;token=16652d03-ac99-437d-a9d8-afd810fb8bff" alt=""><figcaption></figcaption></figure>

As you click the finish button, you'll find your campaign in the campaign list like the above.

### Search a Campaign

To refine or narrow down the campaign list, you can use the search option. Input the campaign related query in the **Search** bar, then from the **Date Range** field, apply a filter.

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FhElFPtMZjXYjDDMBT5q2%2Fuploads%2FSRPKLoMwQwCiRReBVbbd%2FCampaign%20Search.png?alt=media&#x26;token=b2b5e6bc-5318-4c54-92a5-fc866e4b5d8b" alt=""><figcaption></figcaption></figure>

If there is any match, you'd see the campaign in the list.
